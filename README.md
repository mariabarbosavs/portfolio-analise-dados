# ANALISE_DADOS

Repositório criado para organizar meus estudos, práticas e projetos na área de Análise de Dados, com foco em portfólio, resolução de problemas de negócio e evolução para um perfil mais técnico.

## Objetivo
Me preparar para atuar como Analista de Dados Júnior, desenvolvendo:
- raciocínio analítico
- análise de dados com Python e SQL
- visualização de dados
- construção de dashboards
- automação de processos de dados
- base para transição futura para Engenharia de Dados

## Ferramentas do meu ambiente
- Python
- Jupyter Notebook
- VS Code
- SQL
- Power BI
- GitHub
- Excel / Google Sheets
- Apache Hop

## Estrutura do repositório
- `datasets/` → bases de dados para estudo e prática
- `estudos/` → atividades semanais e trilha de aprendizagem
- `notebooks/` → análises em Jupyter Notebook
- `projetos/` → projetos completos para portfólio
- `scripts/` → scripts Python de apoio e automação

## Plano de estudo
Estou seguindo um plano de estudo com foco em:
1. base analítica e organização do ambiente
2. planilhas para análise inicial
3. Python e pandas para exploração e limpeza
4. SQL para perguntas de negócio
5. Power BI para dashboards
6. automação e ETL
7. transição para Engenharia de Dados

## Apache Hop no meu plano
O Apache Hop será usado como ferramenta de apoio para automação e organização de fluxos de dados.

Minha estratégia será:
1. aprender primeiro a fazer análises e tratamentos manualmente
2. depois usar o Apache Hop para automatizar parte do processo
3. comparar o fluxo manual com o fluxo automatizado
