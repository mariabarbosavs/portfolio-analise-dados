# Controle Financeiro para Estudantes com Renda Variável

Este projeto foi desenvolvido no Excel com o objetivo de organizar entradas e saídas financeiras de uma pessoa estudante que não possui renda fixa mensal.

A proposta é simular uma situação comum: receber valores variáveis ao longo do mês, como apoio familiar, presentes ou pequenos recebimentos, e acompanhar como esse dinheiro é utilizado em diferentes categorias de gastos.

## Objetivo do projeto

Criar uma planilha simples e organizada para:

- registrar entradas e saídas financeiras;
- calcular saldo final;
- identificar maior, menor e média de gastos;
- analisar gastos por categoria;
- calcular o percentual de gastos sobre as entradas;
- aplicar formatação visual e condicional para facilitar a leitura dos dados.

## Ferramenta utilizada

- Microsoft Excel

## Recursos aplicados

- Criação de tabelas
- Fórmulas matemáticas
- Função SOMASE
- Função SE
- Funções MÁXIMO, MÍNIMO e MÉDIA
- Cálculo de porcentagem
- Referências de células
- Formatação de moeda
- Formatação condicional
- Organização por abas
- Criação de resumo financeiro

## Estrutura da planilha

A planilha foi organizada em três abas principais:

### 1. Capa

Apresenta o nome do projeto e sua proposta.

### 2. Lançamentos

Contém a base de dados com os registros financeiros simulados.

Colunas utilizadas:

- Data
- Tipo
- Categoria
- Descrição
- Valor
- Forma de Pagamento
- Status
- Mês
- Valor de Saída

### 3. Resumo

Apresenta os principais indicadores financeiros:

- Total de Entradas
- Total de Saídas
- Saldo Final
- Maior Gasto
- Menor Gasto
- Média de Gastos
- Gastos por Categoria
- Percentual de Gastos sobre Entradas

## Observação sobre os dados

Os dados utilizados neste projeto são fictícios e foram criados apenas para fins de estudo e prática em Excel.

## Aprendizados

Durante o desenvolvimento deste projeto, pratiquei a organização de dados em planilhas, criação de fórmulas, separação das informações por abas e construção de um resumo financeiro simples.

Também apliquei formatação visual para melhorar a apresentação do arquivo e facilitar a interpretação dos resultados.
